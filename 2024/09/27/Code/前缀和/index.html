Algorithom:Prefix Sum

Explain

The partial sum is used for caculating sum of many interval data, whose the size is indefine.

Definition

The prefix sum can be understood as the sum of the first n of an array. It can reduce much time when processing.

Example.1

Description

There are n intergers in an array a , now Lvs4nX wanna know the value of $\sum^b_{i=a}a[i]$ .

Pro Sov

use another array sum , which can record the sum of first a in array a . That must be: $sum[i]=\sum_{i=0}^aa[i]$ .
